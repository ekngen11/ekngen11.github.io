Statistical appmore_vert

GitHub Repo

Statistical appclose

This project produced a Java application for web data extraction. The application is targeted at webpages that have information on college hockey teams. It extracts statistics about goals recorded during hockey games over the past few years. All the statistics are consolidated and cached to avoid repeated data extraction, which is time consuming. When launched, the application loads the cached information and provides a user interface that displays the data. The user interface has filtering features for viewing interesting subsets of the data. It can also generate a data matrix for statistical analysis.

Decafclose

This was part of a project to design a simple programming language with syntax simpler than other languages (hence 'decaf'). It is only meant to perform simple functions. The language has a tokenizer that chunks code into tokens, a parser that goes through a hierachy of classes, an interpreter and a simple IDE that allows one to enter the code and run it.

City Learnermore_vert

GitHub Repo

City Learnerclose

In this project, I first developed a model of a city using pygame (Complete with traffic lights, traffic and buildings), and then trained an agent (a car in this case) to navigate the streets of the city while obeying traffic rules. The agent learns to make the right moves using reinforcement learning (receives positive rewards for a 'good' move and negative rewards for a 'bad' move. It (agent) then uses the Q-learning function to find the optimal action policy at a given state.
